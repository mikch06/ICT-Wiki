Mount nfs share

📆 published: 07.09.2015
📆 modified: 23.07.2020
📌 linux, nfs, mount, fstab
1. Setup NFS Client: 
yum -y install nfs-utils
2. Create mountpoint: 
mkdir -p /mnt/nfs/example-share
3. Mount NFS share: 
mount :/example-share /mnt/nfs/example-share
hint: –> df -h
hint: –> mount – show all mounts
4. Add mount to fstab for remounting after reboot: 
    vi /etc/fstab :/example-share /mnt/nfs/
