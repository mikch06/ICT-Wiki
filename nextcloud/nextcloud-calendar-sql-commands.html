Nextcloud calendar:

Select all calendar events older than ... 
SELECT * FROM `oc_calendarobjects` WHERE calendarid = 2 AND lastoccurence >= 1583323200
- calendarid -> e.g. your birthday calendar
- lastoccurence -> timestamp in unixtime
